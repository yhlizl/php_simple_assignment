如果你的 php --ini 顯示 "none"：

方法一：使用簡易啟動腳本
────────────────────────────────────

方法二：手動啟動（一行命令）
────────────────────────────────────

在專案資料夾開啟命令提示字元，執行：

php -d extension=curl -S localhost:8001

然後開啟瀏覽器：
http://localhost:8001/index.html

Q1: 還是顯示 curl_init 錯誤？
────────────────────────────────────
A: 確認使用 start_php_simple.bat 啟動
   或手動加上 -d extension=curl 參數


Q2: API Key 無效？
────────────────────────────────────
A: 前往 https://platform.openai.com/api-keys
   建立新的 API Key 並更新 my_prompt.php


Q3: 網頁打不開？
────────────────────────────────────
A: 確認：
   1. 伺服器已啟動（命令提示字元視窗不要關閉）
   2. 網址正確：http://localhost:8001/index.html
   3. Port 8001 沒有被其他程式佔用


Q4: 想要永久解決？
────────────────────────────────────
A: 執行：啟用_cURL_快速修復.bat
   這會建立 php.ini 並永久啟用 cURL
