========================================
  🔧 修復 cURL 錯誤
========================================

錯誤訊息：
Call to undefined function curl_init()

原因：
Windows 的 PHP 預設沒有啟用 cURL 擴展

========================================

✅ 解決方案（3 種方法）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方法一：啟用 cURL 擴展（推薦）⭐⭐⭐
────────────────────────────────────

1. 找到 php.ini 檔案
   - 開啟命令提示字元
   - 輸入：php --ini
   - 會顯示 php.ini 的位置

2. 用記事本開啟 php.ini

3. 找到這一行（按 Ctrl+F 搜尋）：
   ;extension=curl

4. 移除前面的分號（;）改成：
   extension=curl

5. 儲存檔案

6. 重新啟動 PHP 伺服器
   - 關閉命令提示字元（Ctrl+C）
   - 重新執行：php -S localhost:8001

7. 測試
   - 重新整理網頁
   - 測試 AI 功能


方法二：使用 Python 後端（最簡單）⭐⭐⭐
────────────────────────────────────

Python 版本不需要額外設定！

1. 確認已安裝 Python
   python --version

2. 安裝依賴
   pip install -r requirements.txt

3. 設定 API Key
   - 用記事本開啟 app.py
   - 找到第 20 行
   - 改成：
     OPENAI_API_KEY = 'your-api-key-here'

4. 啟動伺服器
   python app.py

5. 開啟瀏覽器
   http://localhost:8787

6. 修改 config.js（如果需要）
   - 用記事本開啟 config.js
   - 找到第 11 行
   - 改成：
     BACKEND: 'python',


方法三：使用 php.ini-development（快速）⭐⭐
────────────────────────────────────

如果找不到 php.ini：

1. 找到 PHP 安裝目錄
   where php

2. 在該目錄找到 php.ini-development

3. 複製並重新命名為 php.ini
   copy php.ini-development php.ini

4. 用記事本開啟 php.ini

5. 找到並修改：
   ;extension=curl
   改成：
   extension=curl

6. 儲存並重新啟動伺服器

========================================

🔍 檢查 cURL 是否已啟用
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方法一：命令列檢查
────────────────────────────────────
php -m | findstr curl

如果有輸出 "curl"，表示已啟用


方法二：建立測試檔案
────────────────────────────────────
建立 test_curl.php：

<?php
if (function_exists('curl_init')) {
    echo "✅ cURL 已啟用";
} else {
    echo "❌ cURL 未啟用";
}
?>

執行：
php test_curl.php

⚠️ 常見問題
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q1: 找不到 php.ini 檔案？
────────────────────────────────────
A: 執行 php --ini 查看位置
   如果顯示 "Configuration File (php.ini) Path: (none)"
   表示沒有 php.ini，需要從 php.ini-development 複製


Q2: 修改後還是不行？
────────────────────────────────────
A: 確認：
   1. 修改的是正確的 php.ini（用 php --ini 確認）
   2. 已經重新啟動 PHP 伺服器
   3. extension=curl 前面沒有分號


Q3: Windows 找不到 libcurl.dll？
────────────────────────────────────
A: 確認 PHP 安裝目錄中有 ext 資料夾
   裡面應該有 php_curl.dll
   如果沒有，重新下載 PHP（選擇 Thread Safe 版本）


Q4: 還是無法解決？
────────────────────────────────────
A: 使用 Python 後端（方法二）
   或使用 XAMPP（內建 cURL）
