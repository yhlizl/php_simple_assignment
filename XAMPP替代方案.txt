========================================
  XAMPP 安裝不起來的替代方案
========================================

如果 XAMPP 無法安裝，有以下幾種解決方案：

========================================

方案一：使用 Python 版本（最推薦）⭐⭐⭐
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 優點：
- 安裝簡單，只需要 Python
- 不需要 XAMPP
- 效能更好
- 更容易維護

📝 步驟：

1. 安裝 Python
   下載：https://www.python.org/downloads/
   ⚠️ 安裝時記得勾選「Add Python to PATH」

2. 雙擊執行：check_setup.bat
   （會自動檢查並安裝所需套件）

3. 編輯 app.py，設定 API Key
   找到第 20 行，替換成你的 OpenAI API Key

4. 雙擊執行：start_python.bat

5. 瀏覽器開啟：http://localhost:8787

✅ 完成！不需要 XAMPP


========================================

方案二：獨立安裝 PHP（進階）⭐⭐
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

如果堅持要用 PHP，可以不透過 XAMPP 安裝：

📝 步驟：

1. 下載 PHP
   Windows: https://windows.php.net/download/
   選擇「Thread Safe」版本的 ZIP 檔案

2. 解壓縮到 C:\php
   將下載的 ZIP 檔解壓縮到 C:\php\

3. 設定環境變數
   a. 右鍵點擊「本機」→「內容」
   b. 點擊「進階系統設定」
   c. 點擊「環境變數」
   d. 在「系統變數」中找到「Path」
   e. 點擊「編輯」→「新增」
   f. 輸入：C:\php
   g. 點擊「確定」儲存

4. 驗證安裝
   開啟命令提示字元，輸入：
   php -v
   
   如果顯示版本號，表示安裝成功

5. 編輯 my_prompt.php，設定 API Key
   找到第 20 行，替換成你的 OpenAI API Key

6. 修改 index.html
   找到第 606 行，將：
   'http://localhost:8787/api/generate'
   改成：
   'my_prompt.php'

7. 雙擊執行：start_php.bat

8. 瀏覽器開啟：http://localhost:8000/index.html


========================================

方案三：使用 WampServer（XAMPP 替代品）⭐
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

WampServer 是另一個類似 XAMPP 的整合環境

📝 步驟：

1. 下載 WampServer
   https://www.wampserver.com/en/

2. 安裝 WampServer
   按照安裝精靈指示完成安裝

3. 啟動 WampServer
   點擊桌面上的 WampServer 圖示

4. 將專案複製到 www 資料夾
   預設路徑：C:\wamp64\www\

5. 編輯 my_prompt.php，設定 API Key

6. 修改 index.html（同方案二步驟 6）

7. 瀏覽器開啟：
   http://localhost/你的專案資料夾名稱/index.html


========================================

方案四：使用 Laragon（最簡單的 PHP 環境）⭐⭐
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Laragon 是一個輕量級的 PHP 開發環境

📝 步驟：

1. 下載 Laragon
   https://laragon.org/download/

2. 安裝 Laragon
   選擇「Lite」版本即可

3. 啟動 Laragon
   點擊「Start All」

4. 將專案複製到 Laragon 資料夾
   預設路徑：C:\laragon\www\

5. 編輯 my_prompt.php，設定 API Key

6. 修改 index.html（同方案二步驟 6）

7. 瀏覽器開啟：
   http://localhost/你的專案資料夾名稱/index.html

❓ 常見問題

Q: 哪個方案最簡單？
A: 方案一（Python 版本）最簡單，強烈推薦！

Q: 我完全不懂技術，該選哪個？
A: 選方案一（Python 版本），只需要：
   1. 安裝 Python
   2. 雙擊 start_python.bat
   3. 完成！

Q: Python 和 PHP 版本有什麼差別？
A: 功能完全相同，只是後端技術不同
   - Python 版本：更現代、更容易安裝
   - PHP 版本：傳統方式、需要額外設定

Q: 如果 Python 也安裝不起來怎麼辦？
A: 請提供錯誤訊息，我可以協助排除問題
   或者可以考慮使用線上版本（需要額外開發）
