========================================
  手動啟動指南 - 不使用 .bat 檔案
========================================

如果 start_python.bat 或 start_php.bat 無法使用，
可以手動啟動伺服器。

========================================

📁 檔案位置說明
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

所有檔案都在同一個資料夾中，不需要移動！

專案資料夾結構：
📂 你的專案資料夾（例如：C:\Users\你的名字\Desktop\專案）
  ├── index.html          ← 前端網頁
  ├── app.py              ← Python 後端
  ├── my_prompt.php       ← PHP 後端
  ├── config.js           ← 配置檔案
  ├── test_backend.html   ← 測試工具
  ├── start_python.bat    ← Python 啟動腳本
  ├── start_php.bat       ← PHP 啟動腳本
  └── ... 其他檔案

⚠️ 重要：不要移動任何檔案！所有檔案都要在同一個資料夾中！

========================================

方法一：手動啟動 Python 版本（推薦）⭐
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步驟 1：設定 API Key
────────────────────────────────────

1. 用記事本開啟 app.py
2. 找到第 20 行：
   API_KEY = os.getenv('OPENAI_API_KEY', 'your-openai-api-key-here')
3. 將 'your-openai-api-key-here' 替換成你的 API Key
4. 儲存檔案


步驟 2：開啟命令提示字元
────────────────────────────────────

Windows:
1. 按 Win + R
2. 輸入 cmd
3. 按 Enter

或者：
1. 在專案資料夾中
2. 按住 Shift + 右鍵
3. 選擇「在此處開啟命令視窗」或「在此處開啟 PowerShell」


步驟 3：切換到專案資料夾
────────────────────────────────────

在命令提示字元中輸入：
cd "C:\你的專案資料夾路徑"

例如：
cd "C:\Users\John\Desktop\教育輔導工具"

💡 小技巧：
- 可以直接把資料夾拖曳到命令提示字元視窗
- 或在資料夾的網址列輸入 cmd 然後按 Enter


步驟 4：安裝 Python 套件（第一次使用）
────────────────────────────────────

輸入：
pip install -r requirements.txt

等待安裝完成


步驟 5：啟動 Python 伺服器
────────────────────────────────────

輸入：
python app.py

或：
python3 app.py

看到以下訊息表示成功：
🚀 教育輔導建議小工具後端啟動中...
📍 請在瀏覽器開啟: http://localhost:8787


步驟 6：開啟瀏覽器
────────────────────────────────────

1. 開啟瀏覽器（Chrome、Edge 等）
2. 在網址列輸入：http://localhost:8787
3. 按 Enter

✅ 完成！現在可以使用了！


========================================

方法二：手動啟動 PHP 版本
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步驟 1：設定 API Key
────────────────────────────────────

1. 用記事本開啟 my_prompt.php
2. 找到第 28 行：
   $apiKey = getenv('OPENAI_API_KEY') ?: 'your-openai-api-key-here';
3. 將 'your-openai-api-key-here' 替換成你的 API Key
4. 儲存檔案


步驟 2：修改前端配置
────────────────────────────────────

1. 用記事本開啟 config.js
2. 找到第 10 行：
   BACKEND: 'python',
3. 改成：
   BACKEND: 'php',
4. 儲存檔案


步驟 3：開啟命令提示字元並切換到專案資料夾
────────────────────────────────────

（同方法一的步驟 2 和 3）


步驟 4：啟動 PHP 伺服器
────────────────────────────────────

輸入：
php -S localhost:8000

看到以下訊息表示成功：
[Mon Nov 18 22:00:00 2025] PHP 8.x Development Server started


步驟 5：開啟瀏覽器
────────────────────────────────────

1. 開啟瀏覽器
2. 在網址列輸入：http://localhost:8000/index.html
3. 按 Enter

✅ 完成！

❓ 常見問題
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q1: 找不到 python 或 php 命令？
A: 表示沒有安裝或沒有加入 PATH
   - Python: 重新安裝，記得勾選「Add Python to PATH」
   - PHP: 需要先安裝 XAMPP 或獨立安裝 PHP

Q2: Port 已被佔用？
A: 
   - Python: 改用其他 Port，例如：python app.py --port 8788
   - PHP: 改用其他 Port，例如：php -S localhost:8001

Q3: 如何停止伺服器？
A: 在命令提示字元視窗按 Ctrl + C

Q4: 關閉命令提示字元後，網頁就不能用了？
A: 是的！伺服器必須保持運行
   - 不要關閉命令提示字元視窗
   - 或最小化視窗放在背景執行

Q5: 每次都要手動啟動嗎？
A: 是的，每次使用前都要啟動伺服器
   - 可以建立桌面捷徑
   - 或使用 .bat 檔案（如果能用的話）

Q6: 檔案要放在哪個資料夾？
A: 所有檔案都放在同一個資料夾中！
   - 不需要特定位置
   - 可以放在桌面、文件夾等任何地方
   - 重點是所有檔案要在一起

Q7: 可以改資料夾名稱嗎？
A: 可以！資料夾名稱不影響使用
   - 但不要移動裡面的檔案
   - 所有檔案要保持在同一個資料夾

Q8: 需要網路嗎？
A: 需要！
   - 本地伺服器不需要網路
   - 但呼叫 OpenAI API 需要網路連線

💡 建議的資料夾位置
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

推薦位置（選一個即可）：

1. 桌面
   C:\Users\你的名字\Desktop\教育輔導工具

2. 文件夾
   C:\Users\你的名字\Documents\教育輔導工具

3. 自訂位置
   任何你方便的地方都可以

⚠️ 避免的位置：
- ❌ C:\Program Files\（需要管理員權限）
- ❌ 系統資料夾
- ❌ 太深的路徑（例如：C:\A\B\C\D\E\F\...）

🔧 進階：建立桌面捷徑
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Python 版本：
1. 建立新的文字檔案
2. 內容：
   @echo off
   cd "C:\你的專案資料夾路徑"
   python app.py
   pause

3. 另存為：啟動Python.bat
4. 拖曳到桌面

PHP 版本：
1. 建立新的文字檔案
2. 內容：
   @echo off
   cd "C:\你的專案資料夾路徑"
   php -S localhost:8000
   pause

3. 另存為：啟動PHP.bat
4. 拖曳到桌面
