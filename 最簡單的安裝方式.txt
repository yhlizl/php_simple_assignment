========================================
  最簡單的安裝方式（3 步驟完成）
========================================

如果你不熟悉技術，請照著以下步驟做：

========================================

步驟 1：安裝 Python
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打開瀏覽器，前往：
   https://www.python.org/downloads/

2. 點擊黃色的「Download Python」按鈕

3. 下載完成後，雙擊安裝檔案

4. ⚠️ 重要！安裝時請勾選：
   ☑ Add Python to PATH
   
   然後點擊「Install Now」

5. 等待安裝完成，點擊「Close」


========================================

步驟 2：設定 API Key
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 用「記事本」開啟 app.py
   （在檔案上按右鍵 → 開啟檔案 → 記事本）

2. 找到第 20 行，會看到：
   API_KEY = os.getenv('OPENAI_API_KEY', 'your-openai-api-key-here')

3. 將 'your-openai-api-key-here' 替換成你的 OpenAI API Key
   例如：
   API_KEY = os.getenv('OPENAI_API_KEY', 'sk-proj-abc123...')

4. 儲存檔案（Ctrl + S）


========================================

步驟 3：啟動程式
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 雙擊執行：start_python.bat

2. 會出現一個黑色視窗，顯示：
   🚀 教育輔導建議小工具後端啟動中...
   📍 請在瀏覽器開啟: http://localhost:8787

3. 打開瀏覽器（Chrome、Edge 等）

4. 在網址列輸入：
   http://localhost:8787

5. 按 Enter，就可以看到程式了！


========================================

✅ 完成！開始使用
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

現在你可以：
1. 在「孩子能力現況」欄位輸入描述
2. 點擊「🤖 用 AI 產生建議」
3. 等待 AI 自動填入建議
4. 手動調整內容
5. 點擊「✨ 生成整理結果」
6. 點擊「📋 複製整理結果」貼到 Word


========================================

❓ 遇到問題？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

問題 1：雙擊 start_python.bat 後出現錯誤
解決：
- 確認 Python 安裝時有勾選「Add Python to PATH」
- 重新安裝 Python，記得勾選該選項

問題 2：瀏覽器打不開 http://localhost:8787
解決：
- 確認黑色視窗沒有關閉（程式還在執行中）
- 檢查網址是否正確輸入
- 嘗試重新啟動 start_python.bat

問題 3：AI 產生建議失敗
解決：
- 檢查 API Key 是否正確設定
- 確認網路連線正常
- 檢查 OpenAI 帳號是否有額度

問題 4：找不到 app.py
解決：
- 確認你在正確的資料夾中
- 檔案可能被隱藏，開啟「顯示隱藏的檔案」


========================================

🛑 如何停止程式？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 回到黑色視窗（命令提示字元）
2. 按鍵盤上的 Ctrl + C
3. 或直接關閉黑色視窗


========================================

💡 小提示
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 每次使用都要執行 start_python.bat
✅ 黑色視窗不要關閉（程式執行中）
✅ 可以建立桌面捷徑方便啟動
✅ 第一次啟動會比較慢（安裝套件）
✅ 之後啟動就會很快


========================================

📱 建立桌面捷徑（選用）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 在 start_python.bat 上按右鍵
2. 選擇「傳送到」→「桌面（建立捷徑）」
3. 以後就可以從桌面直接啟動


========================================

🎯 總結

只需要 3 個步驟：
1️⃣ 安裝 Python（記得勾選 Add to PATH）
2️⃣ 設定 API Key（編輯 app.py）
3️⃣ 雙擊 start_python.bat

就這麼簡單！
