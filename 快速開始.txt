========================================
  教育輔導建議小工具 - Windows 快速開始
========================================

📌 選擇一種方式啟動：

方法一：Python 版本（推薦）⭐
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 確認已安裝 Python 3.8+
   下載：https://www.python.org/downloads/
   ⚠️ 安裝時記得勾選「Add Python to PATH」

2. 設定 OpenAI API Key
   - 用記事本開啟 app.py
   - 找到第 16 行，將 API Key 替換成你的

3. 雙擊執行：start_python.bat

4. 在瀏覽器開啟：http://localhost:8787


方法二：PHP 版本
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 確認已安裝 PHP 7.4+
   推薦安裝 XAMPP：https://www.apachefriends.org/

2. 設定 OpenAI API Key
   - 用記事本開啟 my_prompt.php
   - 找到第 20 行，將 API Key 替換成你的

3. 修改前端 API 路徑
   - 用記事本開啟 index.html
   - 找到第 606 行
   - 將 'http://localhost:8787/api/generate' 
     改成 'my_prompt.php'

4. 雙擊執行：start_php.bat

5. 在瀏覽器開啟：http://localhost:8000/index.html


========================================

💡 使用提示：

1. 在「孩子能力現況」欄位輸入描述
2. 點擊「🤖 用 AI 產生建議」
3. 等待 AI 自動填入六大向度建議
4. 可手動調整內容
5. 點擊「✨ 生成整理結果」
6. 點擊「📋 複製整理結果」貼到 Word

========================================

❓ 常見問題：

Q: 出現「找不到 Python」錯誤？
A: 請重新安裝 Python，記得勾選「Add Python to PATH」

Q: 出現「找不到 PHP」錯誤？
A: 請安裝 XAMPP 或獨立安裝 PHP

Q: AI 產生失敗？
A: 檢查 API Key 是否正確、網路是否正常

Q: 瀏覽器打不開？
A: 確認伺服器已啟動、網址正確、防火牆未阻擋
