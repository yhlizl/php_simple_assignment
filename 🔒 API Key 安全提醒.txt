========================================
  🔒 API Key 安全提醒
========================================

⚠️ 重要：保護你的 OpenAI API Key

========================================

❌ 什麼是 API Key？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

API Key 是你的 OpenAI 帳號的密鑰，
就像銀行帳號的密碼一樣重要！

格式：sk-proj-xxxxxxxxxxxxxxxx...

========================================

⚠️ 為什麼要保護 API Key？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

如果 API Key 被公開：
❌ 任何人都可以使用你的 API Key
❌ 可能產生大量費用（你要付錢）
❌ 你的 OpenAI 帳號可能被濫用
❌ 可能違反 OpenAI 使用條款

========================================

✅ 如何安全地使用 API Key
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方法一：使用環境變數（最安全）⭐⭐⭐
────────────────────────────────────

Windows:
1. 開啟命令提示字元
2. 輸入：
   setx OPENAI_API_KEY "你的API Key"
3. 重新開啟命令提示字元
4. 啟動伺服器

Mac/Linux:
1. 編輯 ~/.bashrc 或 ~/.zshrc
2. 加入：
   export OPENAI_API_KEY="你的API Key"
3. 執行：source ~/.bashrc
4. 啟動伺服器

優點：
✅ API Key 不會出現在程式碼中
✅ 不會被提交到 Git
✅ 最安全的方式


方法二：直接在程式碼中設定（僅限本地測試）⚠️
────────────────────────────────────

1. 用記事本開啟 my_prompt.php
2. 找到第 29 行
3. 改成：
   $apiKey = getenv('OPENAI_API_KEY') ?: '你的API Key';
4. 儲存檔案

⚠️ 注意：
- 這個方法不安全
- 只適合本地測試
- 絕對不要提交到 Git
- 絕對不要分享給別人


方法三：使用 .env 檔案（推薦）⭐⭐
────────────────────────────────────

1. 在專案資料夾建立 .env 檔案
2. 內容：
   OPENAI_API_KEY=你的API Key
3. 確保 .gitignore 包含 .env
4. 啟動伺服器

優點：
✅ 方便管理
✅ 不會被提交到 Git（如果 .gitignore 設定正確）
✅ 適合開發環境

========================================

🚫 絕對不要做的事
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ 不要把 API Key 寫在程式碼中並提交到 Git
❌ 不要把 API Key 分享給別人
❌ 不要把 API Key 貼在公開的地方（論壇、社群媒體等）
❌ 不要把包含 API Key 的檔案上傳到 GitHub
❌ 不要在截圖中包含 API Key

========================================

✅ 如果 API Key 已經洩漏
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

立即行動：

1. 前往 OpenAI 平台
   https://platform.openai.com/api-keys

2. 刪除洩漏的 API Key
   - 找到該 API Key
   - 點擊「Delete」或「Revoke」
   - 確認刪除

3. 建立新的 API Key
   - 點擊「Create new secret key」
   - 給它一個名稱
   - 複製並妥善保存

4. 更新你的設定
   - 使用新的 API Key
   - 使用環境變數或 .env 檔案
   - 不要再犯同樣的錯誤

5. 檢查使用記錄
   - 前往：https://platform.openai.com/usage
   - 查看是否有異常使用
   - 如有問題，聯絡 OpenAI 支援
