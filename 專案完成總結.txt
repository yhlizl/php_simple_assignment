📝 給客戶的使用說明：

⭐ 最推薦：Python 版本（最簡單）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 安裝 Python（記得勾選 Add to PATH）
2. 編輯 app.py，設定你的 OpenAI API Key
3. 雙擊 start_python.bat 啟動
4. 瀏覽器開啟 http://localhost:8787

詳細步驟請參考：
- 最簡單的安裝方式.txt（純文字版）
- 安裝指南.html（圖文版，用瀏覽器開啟）

如果 XAMPP 安裝不起來：
請參考「XAMPP替代方案.txt」，提供 4 種解決方案：
1. 使用 Python 版本（最推薦）
2. 獨立安裝 PHP
3. 使用 WampServer
4. 使用 Laragon
