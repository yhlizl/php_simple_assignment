【第一次使用，請先閱讀】

1️⃣ 用瀏覽器開啟「安裝指南.html」
   （圖文並茂，最容易理解）

2️⃣ 或閱讀「最簡單的安裝方式.txt」
   （純文字版，3 步驟完成）

【安裝步驟摘要】

步驟 1：安裝 Python
- 前往 https://www.python.org/downloads/
- 下載並安裝
- ⚠️ 記得勾選「Add Python to PATH」

步驟 2：設定 API Key
- 用記事本開啟 app.py
- 找到第 20 行
- 替換成你的 OpenAI API Key

步驟 3：啟動程式
- 雙擊 start_python.bat
- 瀏覽器開啟 http://localhost:8787
- 完成！

- 後端：Flask 3.0.0
- Port：8787
- API 端點：http://localhost:8787/api/generate

- 後端：PHP 內建伺服器
- Port：8000
- API 端點：my_prompt.php
- 需求：PHP 7.4+ with cURL

- 無需編譯或建置
- 支援所有現代瀏覽器

【AI 模型】
- OpenAI GPT-4o-mini
- 需要 OpenAI API Key
